# ⚡ TaskFlow — AI-Powered Task Manager

A personal task manager built with **Streamlit** and **FastMCP**, featuring a stunning dark glassmorphism UI.

## 🚀 Project Overview

TaskFlow is a modern task management application that combines:
- An **MCP server** exposing a SQLite task database via standard tools
- A **Streamlit web app** with natural language chat interface and rich dashboard
- A breathtaking **Dark Glassmorphism + Neon Accent** visual theme

## 🛠️ Tech Stack

- Python 3.10+
- Streamlit
- FastMCP
- SQLite (via aiosqlite)
